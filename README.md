# PHP Web Application

## About

This repository contains a lightweight dynamic web application developed using PHP and MariaDB/MySQL. The development team at Ahmad and Co. worked to modify and deploy the application. The application manages employee data through a browser based interface and demonstrates how PHP communicates with a MariaDB/MySQL database to add, delete, find, and update employee data.

## Features

- Create, read, update, and delete employee records
- Dynamic server-side and client-side web pages
- API endpoints that return data in JSON format
- Python API client examples using the requests module
- Real-time page updates using AJAX

## How It Works

- PHP handles server-side logic and processes incoming requests
- MariaDB/MySQL stores employee data
- The web interface sends requests to PHP scripts
- PHP queries the database and returns results to the browser

## Environment Requirements

- PHP 7.x or 8.x
- MariaDB or MySQL
- Apache or compatible web server
- A configured credentials.php file

## Setup and Configuration

1. Clone this repository
2. Import the sample employees database 
3. Edit credentials.php with your local database connection settings
4. Ensure the appropriate MySQL users have been created
5. Start your web server and find the application in your browser using http
